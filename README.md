# HNC-database-platform

## requirement
+ node v8.5+
+ npm v3.6+

## develop
``` bash
git clone https://github.com/sangle7/HNC-database-platform.git
cd HNC-database-platform
npm install
```
+ 启动前端项目
``` bash
npm run dev:fe 
```
+ 启动服务端
``` bash
npm run dev:ser
```
+ open [localhost:3001](http://localhost:3001)
